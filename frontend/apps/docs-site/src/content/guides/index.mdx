# Guides

Welcome to the Eneo guides section! These practical guides help you implement specific features and use cases that complement the technical documentation in the GitHub repository.

## Available Guides

### [Authentication & OIDC](/guides/authentication)
Learn how to configure single sign-on using OpenID Connect (OIDC) with various identity providers:
- Azure Entra ID (formerly Azure AD)
- Keycloak
- Auth0
- Other OIDC-compliant providers

### [Audit Logging](/guides/audit-logging)
Set up audit logging for compliance and security monitoring:
- Enable audit logs and configure action toggles
- Review access justification flow
- Retention policy and export guidance

### [AI Provider Configuration](/guides/ai-providers)
Connect Eneo to different AI providers and configure them for your organization:
- OpenAI (GPT-4, GPT-3.5)
- Anthropic (Claude)
- Azure OpenAI Service
- Google Gemini
- Local models (Ollama, LM Studio)

### [Deployment](/guides/deployment)
Deploy Eneo in production environments:
- Docker Compose setup
- Kubernetes deployment
- Reverse proxy configuration (Traefik, Nginx)
- SSL/TLS certificates with Let's Encrypt
- Environment configuration best practices

### [Document Processing](/guides/document-processing)
Upload and process documents for AI-powered knowledge bases:
- Supported file formats (PDF, Word, Excel, PowerPoint)
- Web crawling and content extraction
- Chunking strategies for optimal retrieval
- Vector search configuration
- Performance optimization

### [SharePoint Integration](/guides/sharepoint-integration)
Connect Eneo to Microsoft SharePoint and OneDrive for seamless document import:
- Azure AD application registration
- Service account vs. tenant app authentication
- Webhook configuration for real-time sync
- Personal spaces OAuth flow
- Troubleshooting and maintenance
