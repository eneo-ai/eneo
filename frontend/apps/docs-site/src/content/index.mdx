# Eneo

**Democratic AI Platform for the Public Sector**

Eneo is an open-source AI platform designed for public sector organizations to deploy and manage AI assistants while maintaining complete control over data, security, and algorithms.

## Quick Start

Get started with Eneo in minutes:

```bash
# Clone the repository
git clone https://github.com/eneo-ai/eneo.git
cd eneo

# Open in VS Code DevContainer (recommended)
code .
# Click "Reopen in Container" when prompted
```

**Access your instance:**
- Platform: http://localhost:3000
- API Documentation: http://localhost:8123/docs
- Default Login: `user@example.com` / `Password1!`

## Key Features

### ü§ñ AI Assistant Management
Create and customize AI assistants for specific organizational needs with support for multiple AI providers and customizable system prompts.

### üè¢ Collaborative Spaces
Team collaboration within organizations with role-based access control (Admin, Editor, Viewer) and production management capabilities.

### üìö Knowledge Management
Process documents (PDF, Word, PowerPoint, Excel), crawl websites, and enable semantic search using PostgreSQL with pgvector.

### ‚ö° Real-time Capabilities
Streaming chat with Server-Sent Events, WebSocket updates, and background processing for document ingestion and web crawling.

### üåç Multi-language Support
Swedish and English interface with seamless language switching and type-safe translations.
