## Quick Start

Get Eneo running locally in under 5 minutes.

<Tabs items={['Docker Compose', 'DevContainer']}>
  <Tabs.Tab>
    ```bash
    # Clone the repository
    git clone https://github.com/eneo-ai/eneo.git
    cd eneo/docs/deployment/

    # Copy environment templates
    cp env_backend.template env_backend.env
    cp env_frontend.template env_frontend.env
    cp env_db.template env_db.env

    # Add your AI provider key (e.g., OpenAI)
    echo "OPENAI_API_KEY=sk-..." >> env_backend.env

    # Create network and start
    docker network create proxy_tier
    docker compose up -d
    ```

    **Access:** https://localhost (or your configured domain)

    **Default login:** `user@example.com` / `Password1!`
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    # Clone the repository
    git clone https://github.com/eneo-ai/eneo.git
    cd eneo

    # Open in VS Code
    code .
    # Click "Reopen in Container" when prompted
    ```

    The DevContainer includes all dependencies pre-configured for development.

    **Access:**
    - Platform: http://localhost:3000
    - API Docs: http://localhost:8000/docs
  </Tabs.Tab>
</Tabs>

<Callout type="info">
  **Next steps after install:**
  Configure your AI provider in Admin → Models, then create your first Space and Assistant.
  [Full deployment guide →](/guides/deployment)
</Callout>
